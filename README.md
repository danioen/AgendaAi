# AgendaAi
TCC Flutter +Devs2Blu
App de agendamento para um salão de beleza.
IDE = VSCode
Front-end = Flutter

# Objetivos
Objetivo principal: 
Desenvolver um aplicativo móvel, utilizando a linguagem de programação Flutter, que permita aos clientes agendar serviços em salões de beleza de forma prática, rápida e intuitiva, facilitando a gestão de horários e melhorando a comunicação entre o salão e seus clientes.

Objetivos Secundários:

1. Implementar cadastro e login de usuários com autenticação segura.


2. Permitir o agendamento, cancelamento e remarcação de horários diretamente pelo app.


3. Exibir lista de serviços disponíveis com valores e duração.


4. Integrar um calendário visual para facilitar a visualização dos horários disponíveis.


5. Oferecer um painel administrativo para o salão gerenciar serviços, horários e profissionais.


6. Armazenar dados em um banco de dados online garantindo sincronização em tempo real.


7. Aplicar uma interface moderna e responsiva, priorizando a experiência do usuário.
